## Instrukcja instalacji offline

- Pobierz zawartość gałęzi `web`
- Upewnij się że posiadasz zainstalowane środowisko Node.js w wersji **20.12.2** lub nowszej
- Rozpakuj archiwum w dowolnym katalogu
- Przejdź w terminalu do `/web`
- Wywołaj polecenie `npm install`
- Uruchom serwer poprzez `npm run dev`
